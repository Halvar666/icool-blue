# iCool Blue

iCool Blue is a Pale Moon complete theme based on the classic Gradient iCool (Blue) theme by Gradient Studio.

The goal of this project is to preserve the original black-and-blue iCool look while adapting the theme structure, browser chrome, dialogs, menus, scrollbars, and platform-specific behavior for modern Pale Moon.

## Compatibility

- Pale Moon 33.0+
- Tested with Pale Moon 34.x
- Linux and Windows builds are both supported
